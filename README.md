# Mast Magan Restaurant

Premium multi-page restaurant website for Mast Magan Restaurant.

## Pages

- `index.html`
- `about.html`
- `menu.html`
- `booking.html`
- `contact.html`

## Highlights

- Premium dark UI with glassmorphism
- Responsive multi-page layout
- Menu filtering
- Scroll reveals
- Animated counters
- Booking and contact form validation
- Back-to-top button
- Light/dark theme toggle

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- Google Fonts

## Run

Run the local server from the project root:

```bash
npm start
```

Windows launcher:

```bat
start-local.bat
```

If PowerShell blocks `npm`, use:

```bash
cmd /c npm start
```

Then open `http://127.0.0.1:3000/` in your browser.
